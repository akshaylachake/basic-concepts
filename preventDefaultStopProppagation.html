e.preventDefault();

  • Purpose: Prevents the default action associated with the event from being executed.
  • Use Case: When you want to disable the default behavior of an element, such as preventing a form from submitting or a link from navigating.

e.stopPropagation();

  • Purpose: Stops the event from propagating or bubbling up to parent elements.
  • Use Case: When you want an event to be handled only at a specific element and prevent it from being passed on to any parent elements.

e.stopImmediatePropagation();

  • Purpose: Stops the event from propagating (bubbling up) to parent elements and also prevents other event listeners on the same element from being executed.
  • Scope: Affects both the propagation of the event to parent elements and other event listeners on the same element.
