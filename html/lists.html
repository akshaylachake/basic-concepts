Lists

Ordered Lists (<ol>)

  • Ordered lists are used to display a list of items where the order of the items is important. Each item in the list is numbered. 
    
        <ol>
            <li>First item</li>
            <li>Second item</li>
            <li>Third item</li>
        </ol>
    Attributes:
    • type: Specifies the type of numbering (e.g., 1, A, a, I, i).
    • start: Specifies the starting number.
    • reversed: Specifies that the list order should be reversed.
    Example with Attributes: 
    
        <ol type="A" start="3" reversed>
            <li>Third item</li>
            <li>Second item</li>
            <li>First item</li>
        </ol>

Unordered Lists (<ul>)

  • Unordered lists are used to display a list of items where the order of the items is not important. Each item in the list is marked with a bullet point. 
    
        <ul>
            <li>Item one</li>
            <li>Item two</li>
            <li>Item three</li>
        </ul>
    Attributes:
    • type: Specifies the type of bullet (e.g., disc, circle, square).
    Example with Attributes: 
    
        <ul style="list-style-type: circle;">
            <li>Circle bullet item</li>
            <li>Circle bullet item</li>
        </ul>

Definition Lists (<dl>)

  • Definition lists are used to display a list of terms and their definitions. Each term is wrapped in a <dt> element, and each definition is wrapped in a <dd> element. Example: 
    
        <dl>
            <dt>HTML</dt>
            <dd>A markup language for creating web pages.</dd>
            <dt>CSS</dt>
            <dd>A stylesheet language for styling web pages.</dd>
        </dl>
