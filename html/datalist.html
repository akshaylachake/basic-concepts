Datalists

  • The <datalist> element in HTML is used to provide an "autocomplete" feature for an <input> element. It contains a set of <option> elements that represent the available choices. When the user starts typing in the input field, the browser can display a list of options to choose from, based on what the user has typed so far.
  • id The unique identifier for the <datalist> element. This is referenced by the 'list' attribute in the associated <input> element. 
    
        <form>
            <label for="browser">Choose your browser:</label>
            <input list="browsers" id="browser" name="browser" placeholder="Start typing...">
            <datalist id="browsers">
                <option value="Chrome">
                <option value="Firefox">
                <option value="Safari">
                <option value="Edge">
                <option value="Opera">
            </datalist>
            <button type="submit">Submit</button>
        </form>
