Promise

  • A Promise is an object that represents the eventual completion or failure of an asynchronous operation.

A Promise has three states

  • pending — initial state
  • fulfilled — operation succeeded
  • rejected — operation failed

Key Methods

  • .then()
    Runs on success.
  • .catch()
    Runs on error.
  • .finally()
    Runs whether success or error — used for cleanup.
  • Promise.all()
    Runs tasks in parallel → resolves when all succeed. Rejects on first failure.
  • Promise.race()
    Resolves/rejects with the first settled Promise.
  • Promise.allSettled()
    Waits for all Promises to finish regardless of pass/fail.
  • Promise.any()
    Resolves with the first successful Promise (ignores rejections).
