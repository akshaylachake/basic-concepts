Debugging

  • Browser DevTools: F12 or Ctrl+Shift+I (Inspect, Console, Sources, Network, Performance, Memory).
  • IDE Debuggers
  • Console Logging: 
    
        console.log("Debug message:", variable);
  • Breakpoints:
    • Set breakpoints in DevTools or IDE to pause execution and inspect variables.
    • Use debugger; in code to trigger breakpoints.
  • Use ESLint and Prettier to enforce coding standards and catch errors early.
  • Error Handling: 
    
        try {
          riskyOperation();
        } catch (error) {
            console.error("Error:", error);
        }
