Closures

  • A closure is created when a function remembers the variables from its lexical scope, even after the outer function has finished executing.
function outer() { let count = 0; return function inner() { count++; console.log(count); }; } const fn = outer(); fn(); // 1 fn(); // 2

Why are closures useful?

  • Maintain state without using global variables
  • Create private variables
  • Build reusable functions
  • Implement caching and memoization
