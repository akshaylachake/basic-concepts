window

  • Global Object: The 'window' object represents the browser's window. It is the global object in the browser environment, meaning all global variables and functions are properties of the 'window' object.
  • Properties and Methods:
    • window.alert(): Displays an alert dialog.
    • window.setTimeout(): Calls a function or executes a code snippet after a specified delay.
    • window.setInterval(): Repeats a function or code snippet with a fixed time delay between each call.
    • window.location: Provides information about the current URL and methods to navigate to different URLs.
    • window.innerWidth and window.innerHeight: Get the inner width and height of the window.

document

  • DOM Interface: The 'document' object represents the HTML document loaded in the window. It provides methods and properties to interact with the DOM (Document Object Model).
  • Properties and Methods:
    • document.getElementById(): Returns an element by its ID.
    • document.querySelector(): Returns the first element that matches a CSS selector.
    • document.createElement(): Creates a new element.
    • document.body: Represents the 'body' element of the document.
    • document.title: Gets or sets the title of the document.

Key Differences

  1. Scope:
    • window: Represents the entire browser window, including the outer frame and everything inside it.
    • document: Represents the content (HTML) of the webpage loaded in the browser window.
  2. Global Context:
    • window: Global context for the entire web page; any global variables or functions are actually properties of the window object. document: Acts as the entry point to the content of the web page (DOM).
  3. Usage:
    • Use window for actions that involve the browser window itself (e.g., controlling size, opening new windows, setting timeouts).
    • Use document for actions that involve manipulating the HTML content (e.g., selecting elements, modifying content).
