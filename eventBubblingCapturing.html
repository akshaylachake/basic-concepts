Event Bubbling

  • Definition: Event bubbling means that the event starts from the target element and then bubbles up to its ancestors in the DOM hierarchy.
  • Order: From the innermost target element (where the event occurred) to the outer elements.
  • Default Phase: By default, event listeners in JavaScript use event bubbling.

Event Capturing (Trickling)

  • Definition: Event capturing (also known as trickling) means that the event starts from the outermost element and then propagates down to the target element.
  • Order: From the outer elements to the innermost target element.
  • Usage: To use event capturing, the 'addEventListener' method accepts a third boolean parameter which, when set to 'true', makes the event listener execute in the capturing phase.
  • Event Capturing has high priority
