Pseudo-Classes

  • Definition: Pseudo-classes are used to define the special states of an element.
  • Syntax: Pseudo-classes are prefixed with a colon (:).
  • Usage Examples:

    Structural Pseudo-Classes:

    • :root: Selects the document's root element.
    • :nth-child(n): Selects the nth child of a parent element.
    • :nth-last-child(n): Selects the nth child from the end of a parent element.
    • :nth-of-type(n): Selects the nth element of its type among its siblings.
    • :nth-last-of-type(n): Selects the nth element of its type from the end among its siblings.
    • :first-child: Selects the first child of a parent element.
    • :last-child: Selects the last child of a parent element.
    • :first-of-type: Selects the first element of its type among its siblings.
    • :last-of-type: Selects the last element of its type among its siblings.
    • :only-child: Selects an element that is the only child of its parent.
    • :only-of-type: Selects an element that is the only one of its type among its siblings.
    • :empty: Selects elements that have no children (including text nodes).

    State Pseudo-Classes:

    • :hover: Selects elements when the user designates them with a pointing device.
    • :active: Selects elements when they are being activated by the user.
    • :focus: Selects elements that have focus.
    • :focus-visible: Selects elements that have focus and meet specific criteria for being visible.
    • :focus-within: Selects elements that have focus within it or its descendants.
    • :checked: Selects elements that are checked (e.g., checkboxes or radio buttons).
    • :indeterminate: Selects elements that are in an indeterminate state (e.g., checkboxes that are neither checked nor unchecked).
    • :disabled: Selects elements that are disabled.
    • :enabled: Selects elements that are enabled.
    • :read-only: Selects elements that are read-only.
    • :read-write: Selects elements that are editable.
    • :default: Selects elements that are the default among a group of similar elements.
    • :required: Selects elements that are required.
    • :optional: Selects elements that are optional.
    • :valid: Selects elements that have valid values.
    • :invalid: Selects elements that have invalid values.
    • :in-range: Selects elements with values within a specified range.
    • :out-of-range: Selects elements with values outside a specified range.
    • :placeholder-shown: Selects input elements that are showing placeholder text.
    • :target: Selects the current active target element (usually specified by the fragment identifier in the URL).
    • :blank: Selects elements with no user input.
    • :visited: Selects links that the user has already visited.
    • :link: Selects links that the user has not visited.
    • :local-link: Selects links to the current document.

    Miscellaneous Pseudo-Classes:

    • :not(selector): Selects elements that do not match the given selector.
    • :lang(language): Selects elements with a specific language attribute.
    • :any-link: Selects all links.
    • :is(selector): Selects elements that match any selector in a list of selectors.
    • :where(selector): Similar to :is but without specificity.

      /* Changes the background color of a button when hovered */
      button:hover {
          background-color: blue;
      }

      /* Changes the color of a link when it is focused */
      a:focus {
          color: red;
      }

      /* Selects every second list item */
      li:nth-child(2n) {
          background-color: gray;
      }

Pseudo-Elements

  • Definition: Pseudo-elements are used to style specific parts of an element. They allow you to apply styles to parts of the content of an element.
  • Syntax: Pseudo-elements are prefixed with a double colon (::). Older versions of CSS used a single colon (:), which is still supported for backward compatibility.
  • Usage Examples:
    • ::before : Inserts content before the content of an element.
    • ::after : Inserts content after the content of an element.
    • ::first-line : Applies styles to the first line of a block-level element.
    • ::first-letter : Applies styles to the first letter of a block-level element.
    • ::selection : Applies styles to the portion of an element that has been selected by the user.
    • ::placeholder : Styles the placeholder text of an input field.
    • ::marker : Styles the marker box of a list item.
    • ::backdrop : Styles the backdrop of an element, typically used with modal elements.
    • ::slotted(selector) : Selects elements that are distributed to a slot inside a shadow tree.
    • ::part(name) : Selects part of an element defined by a part attribute in Shadow DOM.
    • ::cue : Selects WebVTT cues in a <track> element.
      
        /* Adds content before a paragraph */
        p::before {
            content: "Note: ";
            font-weight: bold;
        }

        /* Adds content after a paragraph */
        p::after {
            content: " Read more...";
            color: blue;
        }

        /* Styles the first line of a paragraph */
        p::first-line {
            font-size: 1.2em;
            font-weight: bold;
        }

        /* Styles the first letter of a paragraph */
        p::first-letter {
            font-size: 2em;
            color: red;
        }
