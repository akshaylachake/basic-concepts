async ... await

  • 'async' and 'await' are keywords in JavaScript that help you work with asynchronous code in a more synchronous fashion.
  • 'async' function: Declares a function that returns an 'AsyncFunction' object. When called, it returns a Promise. If the async function returns a value, the Promise will be resolved with that value. If the async function throws an error, the Promise will be rejected with that error.
  • 'await' expression: Pauses the execution of an async function and waits for the Promise to resolve or reject. It can only be used inside an async function.
